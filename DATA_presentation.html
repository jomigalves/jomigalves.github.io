Introduction

The purpose of this report is to provide an Exploratory Data Analysis (EDA) on data collected from SIHSUS - Brazilian Hospital Information System.

Data from Pernambuco state between 2014 and 2016 was accessed.

Data Setup

Data Download and Conversion

Data was retrieved from ftp.datasus.gov.br from years 2014 to 2016.

Files were downloaded as DBC filetype and converted to CSV.

setwd("C:/HEADS/DATA/DBC_files")

#------------ Download .dbc files from URL------------
url = "ftp://ftp.datasus.gov.br/DISSEMIN/publicos/SIHSUS/MHJ_14_16/"
filenames = getURL(url, ftp.use.epsv = FALSE, dirlistonly = TRUE)
filenames <- strsplit(filenames, "\r\n")
filenames = unlist(filenames)
for (filename in filenames) {
  if( grepl("RDPE", filename)){download.file(paste(url, filename, sep = ""), 
                  paste(getwd(), "/", filename, sep = ""), mode="wb")}}

#------------ Convert .dbc files to .csv ------------
filenames  = list.files()
outputdir <- "C:/HEADS/DATA/CSV_files"
for (filename in filenames) {
  x <- read.dbc(filename)
  file = substr(filename, 1, 8)
  url = paste(outputdir, paste(file, "csv", sep = "."), sep = "/")
  write.csv(x, file = url)}

Improperly structured data

We have analyzed 2 data variables referring to admission date and discharge date (DT_INTER and DT_SAIDA, respectively) to verify if it was properly formatted.

Both these variables were registered as 8-characters long strings, structured as YYYYMMDD.

Both variables were converted into date format. Then we have computed the difference between both values, as to calculate length of stay as a new variable DIAS_CALC.

This analysis used package lubridate.

dataset$DATA_SAIDA <- as.Date(as.character(dataset$DT_SAIDA), format = "%Y%m%d")
dataset$DATA_INTER <- as.Date(as.character(dataset$DT_INTER), format = "%Y%m%d")
dataset$DIAS_CALC <- as.numeric(difftime(dataset$DATA_SAIDA, dataset$DATA_INTER), units = "days")
summary(dataset$DIAS_CALC)

Unreliable data

On the search for unreliable data, we have filtered our dataset for Obstetrics observations as main diagnosis (DIAG_PRINC).

In this setting, we observe that the median length of stay lays at 2 days, with an IRQ of 2 days.

This data is consistent with standard practice. However, a brief look at summary reveals a a maximum of 336 days as length of stay, which is not plausible for common Obstetrics practice.

obst <- dplyr::filter(dataset, grepl("O",DIAG_PRINC))
obst$DIAG_PRINC <- droplevels(obst$DIAG_PRINC)
summary(obst$DIAS_CALC)

Comparison to external data sources

According to the World Health Organization (WHO), the major complications that account for nearly 75% of all maternal deaths are:

  • severe bleeding,
  • infections,
  • high blood pressure during pregnancy,
  • complications from deliver, and
  • unsafe abortion.

In our analysis from Pernambuco state the top 10 causes are in agreement with WHO data.

In the previous table we see cardiogenic/hypovolemic shock as causes of death, which can be the consequence of severe bleeding. Infection could be related with immunization as seen in our data and also with premature rupture of membranes.

There are also different conditions related to hypertension shown.

In our opinion, “spontaneous vertex delivery” should not be considered as a cause of death, so we consider that misclassification may have occurred.

Conclusion

Conclusion

The use of big data in healthcare has the potential to represent the real world, make the connection between several datasets, innovate and improve systems-level understanding. However, with this work, we faced real problems and limitations associated with big data analysis.

To draw conclusions with an impact on clinical practice, data analysis has to be done with caution and create information that is accurate and reliable. In order to improve data quality, data management should be mandatory in health sciences.
